# NEX-ti for NEX2Y

欢迎来到NEX-ti for NEX2Y！快来测测你的属性吧~

## 主要文件说明

- [app.js](app.js)：小程序入口文件，负责全局启动逻辑，比如背景音乐播放。
- [app.json](app.json)：小程序全局配置，控制页面注册和窗口样式。
- [app.wxss](app.wxss)：全局样式文件，定义通用布局和基础视觉风格。
- [pages/start/](pages/start/)：开始页相关文件，负责进入测试前的展示和引导。
- [pages/quiz/](pages/quiz/)：测试核心页面，包含题目展示、结果计算和结果页样式。
- [pages/quiz/data.js](pages/quiz/data.js)：题库数据，定义固定题和随机题内容。
- [pages/quiz/utils.js](pages/quiz/utils.js)：答题计分与结果判定逻辑。
- [pages/quiz/result-data.js](pages/quiz/result-data.js)：各结果类型的文案和展示信息。
- [scripts/result-probability.js](scripts/result-probability.js)：结果概率模拟脚本，用于随机答题统计各结果出现概率。
- [img/](img/)：项目图片资源目录。
